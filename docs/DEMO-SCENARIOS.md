# 🎬 演示场景指南

本指南提供了企业知识库问答系统的完整演示流程，帮助团队理解系统功能。

## 场景一：用户登录（2分钟）

### 目标
演示不同角色的登录和权限差异。

### 步骤
1. 打开 http://localhost
2. 点击右上角「登录」
3. 使用 `admin / admin123` 登录
4. 观察管理员视角：可以看到管理后台入口

### 验证点
- ✅ 登录后显示用户名和角色
- ✅ 管理员可以看到管理后台入口
- ✅ 切换 user1 登录，管理后台入口不可见

## 场景四：AI 智能问答（5分钟）⭐ 重点

### 目标
演示 RAG 智能问答的完整流程，这是系统的核心功能。

### 步骤

#### 简单问答
1. 点击导航栏「AI问答」
2. 在输入框输入：「如何申请年假？」
3. AI 回答：包含年假天数、申请流程、审批步骤
4. 注意回答下方的「来源文档」引用

#### 多轮对话
5. 点击推荐问题：「年假可以跨年使用吗？」
6. AI 基于上下文继续回答

#### 技术问题
7. 新建对话，输入：「系统用了什么技术架构？」
8. AI 回答系统架构信息

#### 销售问题
9. 输入：「客户说价格太贵怎么应对？」
10. AI 回答销售话术策略

### 验证点
- ✅ AI 回答准确引用文档内容
- ✅ 显示来源文档（可点击跳转）
- ✅ 推荐相关问题可点击
- ✅ 对话历史在左侧面板显示
- ✅ 可以新建对话

## 技术架构讲解要点

在演示过程中，可以重点讲解以下技术点：

### 1. RAG 流程（对应场景四）
```
用户提问 → 向量化 → Qdrant检索 → 文档召回 → Prompt构建 → LLM生成 → 答案返回
```

### 2. 数据流
```
前端(Vue3) → Nginx → 后端(Spring Boot) → MySQL/Redis
                            ↓
                    AI服务(FastAPI) → Qdrant/Elasticsearch
```

### 3. 关键技术
- **向量检索**: 使用 Qdrant 存储文档向量，支持语义搜索
- **全文检索**: 使用 Elasticsearch 支持关键词搜索
- **缓存策略**: 使用 Redis 缓存热门数据
- **AI生成**: 支持 OpenAI/Anthropic/模拟三种模式
